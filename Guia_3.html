3.Intervalos de Confianza: "Interval that contains an unknown quantity with a given frequency", All of Statistics, Wasserman.

a. Implemente una función intervalo.mu.asin que tenga por input el nivel $1-\alpha$ y un conjunto de datos $x_1,\ldots, x_n$, provenientes de una muestra de $X$, y devuelva el intervalo de confianza de nivel asintótico $1-\alpha$ para $\mu=\mathbb E(X)$.

In [75]:
intervalo.mu.asin<- function(x,nivel)
{ media<- mean(x)
  desvio<-sd(x)
  n<- length(x)
	
  alfa2<- (1-nivel)/2
  zalfa2<- -qnorm(alfa2)
  liminf<-media-zalfa2*desvio/sqrt(n)
  limsup<-media+zalfa2*desvio/sqrt(n)
	
  list=c(liminf,limsup)
  return(list)
}

c. Algo de Bootstrap: Implemente una función intervalo.mediana.boot que tenga por input el nivel $1-\alpha$ y un conjunto de datos $x_1,\ldots, x_n$, provenientes de una muestra de $X$, y devuelva el intervalo de confianza asintótico $1-\alpha$ para la mediana de $X$. Para ello, asuma que la mediana muestral es asintóticamente normal y estime el desvío del estimador mediante remuestreo con $Nboot=100$.

In [72]:
intervalo.mediana.boot<- function(x,nivel)
{
    Nboot<- 100
# Estimacion bootstrap del desvio de la mediana
# Remuestreo
   mediana_boot<-rep(NA,Nboot)
   for (i in 1:Nboot)
   {
       x_boot<-sample(x,length(x),replace=TRUE)
       mediana_boot[i]<-median(x_boot)
   }

#estimacion boot del desvio
  sd_mediana<-sd(mediana_boot)
  alfa2<- (1-nivel)/2
  zalfa2<- -qnorm(alfa2)

# Calculamos intervalo de confianza para la mediana de x
# usando  desvio del estimador estimado con bootstrap 
lininf<-median(x)-zalfa2*sd_mediana
limsup<-median(x)+zalfa2*sd_mediana
intervalo<-c(lininf,limsup)
return(intervalo)
}
